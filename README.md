# 村田鉄筋㈱ 見積データベース

## プロジェクト概要
- **アプリ名**: 村田鉄筋㈱ 見積データベース
- **目的**: 過去の見積情報をデータベース化し、元請け別・建物構造別・単価別・受注率などを簡単に確認できるようにする
- **主な機能**: 見積データのCRUD・集計ダッシュボード・グラフ表示・絞り込み・CSV/PDF出力・権限管理

## URL
- **ローカル開発**: http://localhost:3000
- **公開URL (sandbox)**: 起動中サンドボックスのGetServiceUrlで取得
- **本番URL (Cloudflare Pages)**: デプロイ後に追記

## テストアカウント
| ロール   | ユーザー名 | パスワード  | 権限                                       |
| -------- | ---------- | ----------- | ------------------------------------------ |
| 管理者   | admin      | admin123    | 全データの閲覧・登録・編集・削除・出力     |
| 一般ユーザー | user   | user123     | 閲覧・登録・編集 (削除不可)                |

## 主要機能 (実装済み)

### 1. 認証・権限管理
- ログイン画面 (`/login`)
- セッショントークン認証 (HMAC-SHA256ベース)
- 管理者 / 一般ユーザーの2ロール
- 削除は管理者のみ実行可能

### 2. 見積データ管理 (`/estimates`)
- 一覧表示 (検索・並び替え・絞り込み)
- 新規登録 (`/estimates/new`)
- 編集 (`/estimates/:id`)
- 削除 (管理者のみ)
- 自動計算: 単価 = 見積金額 ÷ 数量 ÷ 1000

#### 登録項目
- **基本**: 見積番号、見積日、見積担当者
- **元請け・現場**: 元請け会社名、現場名、工事場所、元請け担当者名、連絡先
- **建物**: 構造 (RC造/S造/SRC造/壁構造/ラーメン構造/木造基礎・自由入力可)、用途
- **数量・金額**: 鉄筋数量(t)、見積金額(円)、単価(円/kg)、材料区分(材工/手間請け/支給材)、予想実行単価、利益見込み
- **工期**: 工期、着工予定日、加工開始予定日、難易度、現場担当予定者、再見積有無
- **結果**: 結果 (受注/失注/未定)、失注理由、受注日、競合会社名、備考

### 3. 集計ダッシュボード (`/dashboard`)
- 全体集計カード (総見積件数、受注件数、失注件数、未定件数、受注率、総見積金額、受注金額合計、平均単価、平均数量)
- 月別見積件数・受注件数・失注件数グラフ (Chart.js)
- 失注理由別ドーナツチャート
- 構造別見積件数バーチャート
- 単価帯別受注率バーチャート
- 元請けTOP10サマリ表

### 4. 集計画面
- **元請け別集計** (`/stats/clients`): 受注率・見積金額の横棒グラフ + 表
- **構造別集計** (`/stats/structure`): 件数(棒)と受注率(折れ線)の複合グラフ + 表
- **単価別集計** (`/stats/price`): 単価帯(〜80/81-90/91-100/101-110/111〜)別の受注率バー + 表
- **失注理由別集計** (`/stats/lost`): 円グラフ＋棒グラフ + 件数表

### 5. 絞り込み機能
全画面共通の絞り込みパネル:
- 見積期間 (開始日・終了日)
- 元請け会社名 (部分一致)
- 建物構造 / 建物用途 / 材料区分 / 結果
- 見積担当者 / 失注理由
- 単価帯 (下限・上限)
- キーワード検索 (現場名・備考など)

絞り込み結果に応じて一覧と集計がリアルタイムに更新されます。

### 6. CSV / PDF 出力 (`/stats/export`)
- **CSV出力 (Excel対応・BOM付き)**:
  - 見積データ一覧
  - 元請け別集計
  - 構造別集計
  - 単価別集計
  - 失注理由別集計
- **PDF出力 (A4横向き)**:
  - 集計ダッシュボード全体 (全体集計・元請け別TOP10・構造別・単価別・失注理由別)
  - グラフ画像も同梱
  - ※日本語フォントは jsPDF の制約上、英数字表記となります

## API一覧

| メソッド | パス                | 説明                       | 認証 |
| -------- | ------------------- | -------------------------- | ---- |
| POST     | /api/login          | ログイン                   | 不要 |
| GET      | /api/me             | 自身のユーザー情報         | 必要 |
| GET      | /api/estimates      | 見積一覧 (絞り込み・並び替え対応) | 必要 |
| GET      | /api/estimates/:id  | 見積詳細                   | 必要 |
| POST     | /api/estimates      | 新規登録                   | 必要 |
| PUT      | /api/estimates/:id  | 更新                       | 必要 |
| DELETE   | /api/estimates/:id  | 削除                       | 管理者 |
| GET      | /api/stats          | 集計データ (全パターン)    | 必要 |

### 絞り込みパラメータ (GET /api/estimates, GET /api/stats)
`date_from`, `date_to`, `client_name`, `structure`, `building_use`, `material_type`, `result`, `estimator`, `lost_reason`, `price_min`, `price_max`, `search`, `sort`, `order`

### estimatesテーブル
基本27カラム + メタ情報。詳細は `migrations/0001_initial_schema.sql` 参照。

### ストレージ
- **Cloudflare D1** (SQLite) - 全データの永続化
  - ローカル: `.wrangler/state/v3/d1` に自動生成
  - 本番: Cloudflareの分散SQLiteで全デバイスから共有

## ユーザーガイド

1. **ログイン**: ユーザー名・パスワードを入力 (例: admin/admin123)
2. **ダッシュボード**: 全体の集計サマリと主要グラフを確認
3. **新規登録**: サイドバーの「新規登録」→ 見積情報を入力 → 「登録」
   - 数量と見積金額を入力すると単価が自動計算されます
4. **一覧確認**: 「見積データ一覧」で全データを表形式で確認・編集・削除
5. **絞り込み**: 各画面上部の絞り込みパネルで期間や元請けを指定 → 「適用」
6. **集計確認**: サイドバーの集計メニュー (元請け別・構造別・単価別・失注理由別)
7. **出力**: 「CSV / PDF」画面から目的のフォーマットでダウンロード

## 技術スタック

- **フロントエンド**: HTML/CSS/JavaScript (SPA)、Tailwind CSS、Chart.js、jsPDF、Day.js、Axios
- **バックエンド**: Hono (TypeScript) on Cloudflare Workers
- **データベース**: Cloudflare D1 (SQLite互換)
- **デプロイ**: Cloudflare Pages
